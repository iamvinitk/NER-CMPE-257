540 is looking for an API Integration Specialist to help build an API management platform for an emerging project with the DoD (Department of Defense). Our mission is to create performant and API-first solutions for emerging government requirements including: API enablement of DoD systems, Authentication  Authorization requirements, etc. Additionally, you will work with DevSecOps teams to operationalize  enhance environments (Prod and Non-Prod) to support existing integrations and new integrations of various components to the platform, and configure the API management tool (Apigee) to help drive success and implement common behaviors across the platform. Data Mediation, Traffic Management, and Security are all areas of the platform where you will focus to improve the DoD enterprise as a whole, and help the DoD become more efficient and maintainable in the way DoD stakeholders make decisions and actions on critical information.
