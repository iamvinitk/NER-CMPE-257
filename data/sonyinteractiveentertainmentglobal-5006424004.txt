The role will be reporting into the Engineering Manager and will be responsible for designing and developing user interfaces that are engaging, user-friendly, and visually appealing as well as full stack development of web platforms for both internal and external usage. 
 
Responsibilities 
As a UIUX Full Stack Developer, you will be responsible for designing and developing user interfaces that are engaging, user-friendly, and visually appealing. You will work closely with our design and development teams to translate user requirements into seamless and intuitive digital experiences.   

Consistent User Interface: Ensure that the user interface (UIUX) of the pipeline applications aligns with the overall look and feel of the rest of our internal tools. Consistency in design elements, and UX functionality. 
Unified Navigation: Design a unified navigation structure that allows users to easily switch between different office productivity applications within the open-source platform. This can be achieved through a centralized menu or sidebar that provides access to various applications and features.  
Single Sign-On: Implement a single sign-on (SSO) mechanism to enable users to log in once and access multiple office productivity applications seamlessly. Integration with authentication protocols, such as OAuth or SAML, can streamline the login process and enhance user convenience.  
Accessibility Considerations: Ensure that the interface of office productivity applications adheres to accessibility standards and guidelines, making it inclusive and usable for users with disabilities. Consider factors such as keyboard navigation, screen reader compatibility, and appropriate colour contrast.  
Responsive Design: Optimize the interface for different devices and screen sizes. Implement responsive design techniques to ensure that the office productivity applications can be accessed and used seamlessly on desktops, laptops, tablets, and mobile devices.  
User Feedback and Iterative Design: Gather feedback from users throughout the design and development process. Conduct user testing, surveys, and feedback sessions to understand user preferences, pain points, and suggestions for improvement. Use this feedback to iterate and refine the interface design. Requirements: Your expertise in both front-end and back-end development will contribute to the successful implementation of projects.  
Collaborate with the design team to create wireframes, mockups, and interactive prototypes that effectively communicate the user interface and user experience design concepts.  
Develop responsive and user-friendly web applications, ensuring cross-browser compatibility and optimal performance. 
Implement front-end designs using HTML, CSS, and JavaScript frameworks while adhering to industry best practices.  
Design and develop RESTful APIs and back-end services using server-side technologies (such as PythonDjango or Flask). 
Conduct usability testing and gather user feedback to continuously improve the user experience. 
Collaborate with cross-functional teams, including designers, developers, and project managers, to ensure successful project delivery. 
Stay up to date with the latest UIUX trends, technologies, and best practices
