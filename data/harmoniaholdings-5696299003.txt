Harmonia has an exciting opportunity for a highly skilled Senior Data Analytics and Machine Learning Software Developer to join our team.
Job Description:
Serve as a member of a research and development team, provide expert-level software development services to build innovative data analytic solutions for complex domains using Big Data open sources technologies and Machine Learning (ML) algorithms, in an AWS cloud environment.
Requirements:

Expert level skills and experience using Python, Big Data stack, and ML algorithms to develop large data science and data engineering solutions.
Solid experience utilizing AWS cloud services including Glue, EMR, Athena, Lake Formation, Kinesis, MSK, SageMaker, S3, S3 data lake, CloudWatch, RDS, DynamoDB, RedShift, Lambda, IAM, Boto3...
Proficiency using Python-based ML, data engineering, and data visualization libraries including:
Pandas, NumPy, SciPy, MatPlotLib, Seaborn...
PySpark, Spark, Spark Streaming
Apachi Hudi, Spark, Flink, Presto, HADOOP, Hive, Airflow, Parquet, Delta Lake
TensorFlowKeras, PyTorch, Scikit-Learn, XGBoost, etc.
Good understanding of ML and statistical methods including classification, clustering, regression, Time Series Analysis algorithms
Proficient using modern software development tools and environments including VS Code, GitHub, Jupyter notebook, Jira, Anaconda, AWS CLI
Familiarity with Deep Learning models and technologies (CNNs, RNNs, GANs, transformers, pretrained LLMs...)
Knowledge of and experience with Geospatial data and technologies is a plus.

Key Tasks  Responsibilities:

Design and develop data analytic solutions and platforms to solve operational problems such as complex event detection, entity resolutions, etc.
Design, implement, deploy, and maintain software systems applying state-of-the-art Lean Agile and DevSecOps practices and utilizing CICD pipelines in a cloud environment.
Design and implement innovative data analytic capabilities to ingest, transform, process, analyze, and visualize large volumes of batch and streaming data obtained from heterogeneous sources in near real time.

Education  Experience:

7 years of data engineering and ML solution engineering
5 AWS development experience utilizing native cloud services to deliver large data analytic and ML solutions.
Masters degree in CS, Math, Computational Science, or a related field, or the equivalent combination of education and related experience.
Expert level experience developing software with using Python, Java and open-source Big Data stack in AWS cloud environment.
Significant experience providing innovative approaches and applying new technologies to broadly defined tasks and projects.
Proficient verbal and written communication skills to effectively collaborate in a team environment, present, and explain technical concepts, document solutions, prepare and present high-quality briefings.
