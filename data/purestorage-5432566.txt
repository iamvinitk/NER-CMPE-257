What You Will Be Doing
Looking for a Front-End  engineer to play a role in our evolution towards a more cloud-native platform.  As an individual contributor, you will be responsible for end to end UI development (designing, developing, integrating, maintaining, and supporting) of our sales platform. The position will involve evolving and integrating our platform into other UI frameworks and micro frontends.

Front-End design and development for enhancing our platform business flow or remediatingrefactoring application bugs
Create user experience with reusable components and frameworks across our platform to be seamless, intuitive, and pleasant
Design and develop feature rich api driven web applications using HTML5, CSS3, JavaScript and Angular  
Ensure UI availability and ability to rapidly and creatively solve application problems
Build and maintain JS UI Libraries
Proactive with all phases of testing including test planning, functional testing, regression testing and code review
Leverage company and industry best practices and standards
Specify and integrate 3rd part or externally developed web components
Work autonomously and independently as needed

WHAT YOULL NEED TO BRING TO THIS ROLE...

BS degree in Computer Science, Engineering, MIS, or similar relevant field
5 years experience of enterprise front-end software development
A deep understanding of JavaScript and experience building web application front ends with modern frameworks such as Angular
Hands-on experience with frontend troubleshooting (memory leaks, performance, security, content-header, etc)
Experience with Amazon Web Services and cloud native application development using AWS, Github, Spinnaker (container based build and deployment)
